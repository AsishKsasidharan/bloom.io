Bay of Bengal Lab for

and Modelling

We investigate the physical, chemical, and biological dynamics of the Bay of Bengal using in-situ observations, satellite remote sensing, Argo profiling floats, and regional ocean models (CROCO). Our goal is to advance fundamental understanding of monsoon–ocean coupling and contribute to improved prediction of the Indian Ocean system.

Ocean modelling tutorials
Hands-on guides for ocean scientists
Step-by-step tutorials covering CROCO regional modelling, Argo data analysis, Python tools, and reanalysis diagnostics. Suitable for graduate students and early-career researchers.
Featured series: Setting up CROCO for the Bay of Bengal
6-part guide · Beginner to intermediate · Python & Fortran
01
Introduction to CROCO: architecture, compilation, and grid generation
Install CROCO, configure the Makefile for a Linux HPC environment, and generate your first idealized domain using CROCO_TOOLS in MATLAB or Python.
Beginner CROCO Python
02
Preparing atmospheric and boundary forcing for a Bay of Bengal configuration
Download and regrid ERA5 atmospheric forcing and GLORYS12 ocean boundary conditions. Handle Ganges–Brahmaputra river runoff as a surface freshwater flux.
Beginner ERA5 GLORYS12 xarray
03
Running and monitoring a CROCO simulation on HPC clusters
Submit CROCO jobs on PBS/SLURM, monitor progress, handle restart files, and set up basic diagnostics. Covers common runtime errors and how to fix them.
Intermediate HPC SLURM CROCO
04
Post-processing CROCO output with xarray and Python: temperature, salinity, and MLD
Load CROCO NetCDF output using xarray, compute mixed layer depth using density and temperature criteria, and plot T-S diagrams and hovmöller plots.
Intermediate xarray matplotlib cmocean
05
Working with Argo float data for the Bay of Bengal: quality control and climatology
Download Argo profiles from the GDAC, apply QC flags, bin to a seasonal climatology, and compare against WOA18 and CROCO model output.
Beginner Argo argopy Python
06
Mixed layer heat budget analysis using reanalysis and observations
Compute net heat flux, entrainment, horizontal advection, and diffusion terms. Attribute seasonal MLD variability to individual forcing components.
Advanced ERA5 CROCO numpy
07
Satellite altimetry analysis: sea level anomaly and eddy detection in the BoB
Download and process CMEMS DUACS gridded SLA data. Identify mesoscale eddies using the Okubo-Weiss parameter and track their lifecycle over the Bay of Bengal.
Intermediate CMEMS cartopy scipy
